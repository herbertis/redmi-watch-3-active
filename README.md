# ⌚ Redmi Watch 3 Active - Product Landing Page

[Preview do Projeto](https://herbertis.github.io/redmi-watch-3-active-page/)

## 💻 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Landing Page de produto para o Redmi Watch 3 Active**, inspirada nos padrões visuais utilizados em páginas oficiais de lançamento de produtos da Xiaomi e vitrines de e-commerce.

A página foi desenvolvida para **apresentação e venda do produto nos canais oficiais da Xiaomi Brasil**, incluindo vitrines digitais utilizadas em marketplaces parceiros.

O objetivo foi criar uma interface moderna, visualmente atrativa e altamente responsiva, destacando os principais recursos do smartwatch através de **hierarquia visual clara, seções visuais imersivas e storytelling focado nos benefícios do produto**.

A landing page foi construída com foco em **performance, design responsivo e experiência do usuário**, simulando o padrão de páginas utilizadas por grandes marcas de tecnologia.

---

## 🚀 Principais Funcionalidades (Features)

* **Hero Section de Produto:** Apresentação inicial destacando o design e os principais diferenciais do smartwatch.
* **Storytelling Visual do Produto:** Seções dedicadas para apresentar funcionalidades como monitoramento de saúde, bateria e recursos esportivos.
* **Layout Inspirado em Product Pages de Tech Brands:** Estrutura semelhante às páginas oficiais utilizadas por grandes fabricantes de tecnologia.
* **Design Responsivo:** Interface adaptável para **Mobile, Tablet e Desktop**.
* **Seções Visuais de Recursos:** Apresentação clara das funcionalidades e especificações do smartwatch.
* **Performance Front-end:** Estrutura leve e otimizada sem dependência de frameworks pesados.

---

## 🛠️ Tecnologias Utilizadas

Neste projeto, foi adotada uma abordagem **clean e performática**, priorizando controle total da interface e carregamento rápido.

* **HTML5:** Estrutura semântica e organização do conteúdo.
* **CSS3:** Flexbox, CSS Grid, responsividade, animações e variáveis CSS.
* **JavaScript (Vanilla):** Manipulação de DOM e interações da interface.
* **Responsive Design:** Media Queries para adaptação em múltiplos dispositivos.

---

## 🎨 Abordagem de Design (UI/UX)

Como profissional que une **Design e Desenvolvimento Front-end**, o foco desta landing page foi criar uma experiência visual que valorize o produto e facilite a navegação do usuário.

Principais princípios aplicados:

1. **Hierarquia Visual:** Destaque progressivo das funcionalidades do smartwatch.
2. **Design Orientado à Escaneabilidade:** Conteúdo estruturado para leitura rápida.
3. **Experiência Mobile First:** Interface pensada inicialmente para dispositivos móveis.
4. **Estética Tecnológica:** Visual alinhado com padrões de marcas globais de tecnologia.

---

## 📈 O Impacto (Business Value)

A construção desta landing page demonstra como **design e front-end podem aumentar a percepção de valor de produtos tecnológicos em ambientes de e-commerce**.

Principais benefícios desta abordagem:

1. **Valorização do Produto:** Layout moderno que reforça o posicionamento premium do smartwatch.
2. **Melhoria da Experiência do Usuário:** Navegação clara e intuitiva.
3. **Engajamento Visual:** Seções visuais que incentivam a exploração da página.
4. **Compatibilidade com Marketplaces:** Estrutura preparada para integração com vitrines digitais e páginas de produto.
